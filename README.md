# React Assignment Project

## Overview
This project was created as part of my web development assignment using React.  
The goal of the project was to build a simple interface and practice working with Git, GitHub, and React components.

---

## Tools & Technologies Used
- React (Vite)
- TypeScript
- Bootstrap
- Material UI (MUI)
- Git & GitHub

---

## Project Setup Steps

### 1. Creating the Project
I created the React project using Vite inside my local development folder:
C:\dev\react-projects\my-app

I made sure to move all work away from Desktop/OneDrive to avoid syncing and Git issues.

---

### 2. Installing Dependencies
I installed the required libraries:

- Bootstrap for styling
- Material UI for the slider component

Commands used:npm install bootstrap
npm install @mui/material @emotion/react @emotion/styled

---

### 3. Setting Up Git & GitHub
I initialized Git inside the project folder and connected it to GitHub:

- git init
- git add .
- git commit
- git remote add origin
- git push

My project is now uploaded to my GitHub repository.

# React + Tailwind CSS Project

## 📌 Project Overview
This project is a React application where I migrated from Bootstrap to Tailwind CSS and built a simple login page using utility-first styling.

---

## 🛠️ Steps Performed

### 1. Removed Bootstrap
- Uninstalled Bootstrap from the project
- Removed Bootstrap imports from the codebase

### 2. Installed Tailwind CSS
- Installed Tailwind, PostCSS, and Autoprefixer
- Configured `tailwind.config.js`
- Set up `postcss.config.cjs`
- Added Tailwind directives in `index.css`

### 3. Fixed Configuration Issues
- Resolved PostCSS module format errors
- Fixed Tailwind v4 compatibility issue by installing `@tailwindcss/postcss`

### 4. Built Login Page
- Created a simple login UI
- Styled using Tailwind utility classes
- Used flexbox, spacing, borders, and hover effects

---
